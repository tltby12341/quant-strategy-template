## 写入规则

进入本文件的条件：
1. 至少 2 个独立回测样本一致支持该结论
2. 每条铁律必须标注 **验证条件**（初始资金、测试区间、样本量）
3. 环境依赖必须显式声明（牛市/震荡市/熊市结论可能相反）
4. 反例或推翻性数据出现时，立即降级或移除，不保留"曾经正确"的残留

新证据来源：`performance-matrix.md` + `history.md` 的法医结论。

## 模板

### DON'Ts（绝对禁忌）

```
### N. 禁止 XXX

**陷阱**: 常见直觉或叙事
**数据**: 具体回测对照（版本号、样本量、收益差）
**真相**: 为什么直觉错了
**环境依赖**: 何种市场环境下此结论成立
```

### DOs（必选原则）

```
### N. XXX

**法则**: 具体可执行规则
**验证**: 支撑的回测样本
**原理**: 为什么有效
**环境依赖**: 适用范围
```
